# ReactNativePlayground
Basic React Native Tutorial

## How To Install
```
git clone https://github.com/randyviandaputra/ReactNativePlayground.git
cd ReactNativePlayground
yarn install
yarn run android (run on android)
yarn run ios (run on ios)

```
